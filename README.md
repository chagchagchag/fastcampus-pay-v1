## fastcampuspay-v1
ddd 프로젝트의 v1 버전입니다.
- Spring Boot Web
- Spring Data JPA
- mysql

Servlet Stack 기반이며, v2 버전에서 webflux 기반으로 전환 예정입니다. DDD 프로젝트로는 첫 개발이여서 삑사리날 것 같아서 Servlet Stack 버전을 선택했습니다. 흑흑...
<br/>

v1 버전을 빠르게 끝낸 후 v2 버전을 webflux 기반의 프로젝트로 빠르게 시작해볼 예정입니다.<br/>
